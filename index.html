Qutum

Qutum, short for quantum, is a simple, efficient, powerful and layered programming language.

Project hosted here.

Feature

Layer 1

  • Quite concise semantic and syntax
  • No side effect pure functional
  • Data flow
  • Pure object oriented
  • Memory safety
  • Closure function
  • Named parameter
  • Default parameter
  • No null nor nil
  • Multiple inheritence without name conflict
  • Static type
  • Type inference
  • Polymorphic generic type
  • Duck typing
  • Declarative concurrence

Layer 2

  • C backend
  • WebAssembly backend
  • Lua backend
  • Aggressive memory management
  • Global garbage collection
  • Integer optimization
  • Tail recursion elimination

A compiler is under development, you could try an old incomplete graphical demo on HTML5.

Qutum

Qutum，quantum简写，是一个简单高效强大的分层的程序语言。

项目站点在这里。

特性

层1

  • 语义和语法相当简洁
  • 纯函数式无副作用
  • 数据流
  • 纯面向对象
  • 内存安全
  • closure函数
  • 命名参数
  • 默认参数
  • 无null无nil
  • 多重继承无名称冲突
  • 静态类型
  • 类型推导
  • 多态泛型
  • duck类型

  • Lua后端
  • 积极的内存管理
  • 全局垃圾回收
